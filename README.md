# Suggest
# Installation


```
    > git clone https://github.com/mirchenko/suggest.git yourfolder
    > cd yourfolder
 ```
 
Set up your mongo credentials in `./mongo-connect` `connection` field.

 ```
    > yarn server-build
    > node server.js
```

# Usage

```
    > yarn start
```
Open your browser and navigate to [http://localhost:8080](http://localhost:8080)
